# LEXPRO — клиент «Правовые запросы»

Отдельное приложение на том же стеке (Vite, React, Tailwind 4), что и основной фронт **Legal Boards**. Использует те же CSS-токены бренда (`brand`, палитра slate) из общего подхода к теме.

## Связь с бэкендом

Без изменений бэкенда: запросы отображаются как **задачи выбранной доски** в текущем рабочем пространстве.

1. Запустите API Legal Boards (по умолчанию `http://localhost:5004`).
2. При необходимости задайте адрес API в `.env`:

```bash
VITE_API_URL=http://localhost:5004/api
```

3. Войдите под учётной записью того же сервера (`POST /api/auth/login`).

В сайдбаре выберите **пространство** и **доску** — список слева подтягивается через `GET /api/tasks/board/:boardId`, карточка запроса — через `GET /api/tasks/:id`, комментарии во вкладке «Чат» — `POST /api/tasks/:id/comments`.

## Запуск

Из каталога `lexpro-front`:

```bash
pnpm install
pnpm dev
```

Откроется интерфейс на порту **5174**, чтобы не пересекаться с основным приложением (`5173`).

Сборка:

```bash
pnpm build
pnpm preview
```

## Концепция UI

Макет с узкой колонкой «Мои запросы» и областью деталей с вкладками повторяет ваш эскиз; стилистика выровнена с основным продуктом (бирюзовый акцент `brand`, нейтральные серые фоны и рамки).
